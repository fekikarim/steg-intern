# 1. Introduction

## 1.1 Purpose

This document identifies and analyzes the main operational, organizational, and technical pain points observed in the current internship management and administrative processes at STEG.

The objective is to understand the difficulties faced by each stakeholder and define the problems that the future digital platform must solve.

This analysis serves as a foundation for:

- Requirements definition
- System design decisions
- Feature prioritization
- Workflow optimization
- Digital transformation strategy

---

# 2. Current Situation Overview

Currently, internship management at STEG relies heavily on manual processes involving:

- Paper documents
- Physical signatures
- Manual verification
- Email exchanges
- Independent Excel files
- Multiple communication channels
- Human coordination between departments

Although these processes allow internship operations to continue, they create several inefficiencies that impact productivity, traceability, and user experience.

---

# 3. Main Pain Points Identification

---

# 3.1 Administrative Process Complexity

## Description

Internship administration requires coordination between multiple actors:

- Candidates
- HR department
- Internship supervisors
- Departments
- Financial services
- Management

Each actor handles different parts of the process, making coordination complex.

## Observed Issues

- Multiple manual interventions
- Lack of centralized information
- Difficulty tracking application progress
- Repeated data entry
- Dependency on human communication

## Impact

| Impact Area | Consequence |
|-|-|
| Time | Longer processing delays |
| Productivity | Increased administrative workload |
| Quality | Higher probability of errors |
| Experience | Poor user satisfaction |

---

# 3.2 Paper-Based Document Management

## Description

Internship management involves many documents:

- Internship requests
- CVs
- Motivation letters
- University agreements
- Administrative forms
- Internship conventions
- Certificates

These documents are often managed manually.

## Observed Issues

- Physical document storage
- Risk of document loss
- Difficult document retrieval
- No version management
- Slow validation process

## Impact

The absence of centralized digital document management leads to:

- Reduced accessibility
- Administrative delays
- Lack of traceability
- Increased operational costs

---

# 3.3 Lack of Real-Time Internship Tracking

## Description

There is currently no unified system allowing stakeholders to monitor internship progress.

## Observed Issues

Stakeholders cannot easily know:

- Current internship status
- Pending validations
- Completed tasks
- Missing documents
- Supervisor feedback

## Impact

This creates:

- Communication overhead
- Delayed actions
- Difficulty measuring internship progress

---

# 3.4 Weak Workflow Visibility

## Description

Administrative workflows involve multiple approval and validation steps.

Examples:

- Internship application validation
- Document approval
- Financial validation
- Internship completion process

## Observed Issues

- Workflow states are not centralized
- Difficult identification of responsible actors
- No automatic notifications
- Manual follow-up required

## Impact

Consequences include:

- Bottlenecks
- Forgotten actions
- Longer processing cycles

---

# 3.5 Communication Fragmentation

## Description

Communication between stakeholders happens through multiple channels:

- Emails
- Phone calls
- Physical meetings
- Messaging applications

## Observed Issues

- Information distributed across different platforms
- No communication history
- Difficult follow-up
- Important information may be missed

## Impact

This reduces:

- Transparency
- Collaboration efficiency
- Traceability

---

# 3.6 Absence of Internship Digital Companion

## Description

Interns currently lack a dedicated digital environment during their internship period.

## Missing Capabilities

No centralized solution for:

- Daily journal
- Task management
- Deliverable submission
- Supervisor feedback
- Progress tracking
- Notifications

## Impact

Interns and supervisors must rely on manual methods, reducing internship monitoring quality.

---

# 3.7 Limited Data Centralization

## Description

Internship information is distributed between different sources.

Examples:

- Paper archives
- Personal files
- Excel documents
- Emails

## Observed Issues

Difficulty accessing:

- Candidate history
- Previous internships
- Department statistics
- Supervisor assignments

## Impact

This prevents efficient:

- Decision making
- Reporting
- Analysis

---

# 3.8 Lack of Advanced Reporting and Analytics

## Description

Management requires visibility over internship activities.

## Current Limitations

Difficulty obtaining:

- Number of interns
- Internship distribution by department
- Historical statistics
- Financial summaries
- Performance indicators

## Impact

Strategic decisions are harder due to lack of reliable dashboards.

---

# 3.9 Manual Financial Management

## Description

Internship payments and indemnities require administrative processing.

## Observed Issues

- Manual verification
- Lack of payment status tracking
- Limited integration with internship information

## Impact

Possible consequences:

- Processing delays
- Data inconsistency
- Administrative workload

---

# 3.10 Security and Traceability Limitations

## Description

Administrative systems require strong control over sensitive information.

Examples:

- Personal candidate information
- Documents
- Financial data
- Validation decisions

## Observed Issues

- Limited access control
- No centralized audit history
- Difficult tracking of modifications

## Impact

Risks:

- Unauthorized access
- Data integrity issues
- Lack of accountability

# 8. Conclusion

The current internship management process presents several operational limitations mainly caused by manual handling, fragmented information, and lack of automation.

The STEG Smart Internship & Administrative Workflow Platform aims to transform this process into a centralized, secure, and intelligent digital ecosystem.

By addressing these pain points, the solution will improve:

- Administrative efficiency
- Internship monitoring quality
- Stakeholder collaboration
- Data reliability
- Strategic decision making
